# Template

## 📝 오늘 배운 내용 요약

1. 데이터 시각화
: 표와 숫자만으로는 발견하기 어려운 데이터의 특징을 눈으로 쉽게 확인 할 수 있도록 도와줌

2. 데이터 시각화가 필요한 이유
- 데이터 분포 확인 가능 (히스토그램,박스플롯 등)
- 집단별 비교가 가능 (막대그래프 등)
- 시간에 따른 변화 확인 가능 (선 그래프 등)
- 이상치 발견 가능 (산점도,박스플롯 등)
- 변수 간 관계를 확인 가능 (산점도,히트맵 등)
- 분석 결과를 효과적으로 전달 가능
3. 데이터 시각화의 주요 목적
4. 주요 데이터 시각화 라이브러리
- **Matplotlib **
- 파이썬의 대표적인 시각화 라이브러리
- 다양한 종류의 그래프를 만들 수 있음
- 그래프의 제목, 축, 범례, 크기 등을 세부적으로 설정 가능
- 다른 시각화 라이브러리의 기반으로 활용됨
- 자유도가 높지만 설정해야 할 항목이 비교적 많음
- 기본적인 그래프를 직접 만들 때
- 그래프의 세부 요소를 자유롭게 설정할 때
- 여러 그래프를 하나의 화면에 배치할 때
- 그래프의 크기, 축, 제목, 범례 등을 자세히 조정할 때


- **Seaborn **
- Matplotlib을 기반으로 동작
- 기본 그래프 디자인이 깔끔함
- 데이터프레임의 컬럼명을 직접 지정할 수 있음
- 범주별 비교와 통계적 시각화에 편리함
- 여러 변수의 관계를 간단하게 표현할 수 있음
- 데이터프레임을 이용해 빠르게 그래프를 만들 때
- 범주별 데이터 차이를 비교할 때
- 데이터의 분포와 이상치를 확인할 때
- 여러 변수의 관계와 상관관계를 분석할 때
- 기본적으로 보기 좋은 통계 그래프가 필요할 때
```python
Matplotlib
└── 그래프를 만드는 기본 기반
└── Seaborn
└── 통계 그래프를 더 쉽고 보기 좋게 표현
```

5. 좋은 데이터 시각화의 조건
- 분석 목적에 맞는 그래프를 선택
- 범주 비교 → 막대그래프
- 시간에 따른 변화 → 선그래프
- 데이터 분포 → 히스토그램
- 이상치 확인 → 박스플롯
- 변수 간 관계 → 산점도
- 제목과 축 이름을 명확하게 작성
- 하나의 그래프에 너무 많은 내용을 넣지 X
```python
시각화 목적 설정
↓
분석할 변수 선택
↓
적절한 그래프 선택
↓
그래프 생성
↓
제목·축·범례 설정
↓
패턴과 관계 해석
↓
인사이트 도출
```
